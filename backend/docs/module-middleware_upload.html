Module

middleware/upload

Middleware konfigurasi upload file menggunakan Multer. Middleware ini mengatur lokasi penyimpanan file, penamaan file berdasarkan nama produk, serta melakukan validasi tipe file yang diizinkan.

View Source src/middlewares/upload.js, line 1

Examples
// Menggunakan middleware upload di route produk:
const upload = require("../middleware/upload");
router.post("/add", upload.single("produk_gambar"), insertProduk);

// File akan tersimpan di folder "uploads/" dengan nama sesuai produk.
// Response sukses:
{
  "message": "Sukses upload gambar",
  "result": {
    "filename": "roti-manis.jpg",
    "path": "/uploads/roti-manis.jpg"
  }
}
// Response error jika format file tidak didukung:
{
  "message": "Format file tidak didukung",
  "result": null
}
